# St Mary's Church Handforth

St Mary's is a small, friendly Methodist church with a big heart, situated in Handforth village, East Cheshire. We are part of the Bramhall & Wythenshawe Methodist Circuit in the North West of England District.

## Website Redesign (2026)

### Brand Identity
- **Primary Green**: #2E6E4A (Amazon)
- **Light Green**: #8FBF9A (Summer Green)
- **Accent Gold**: #F2C77D (Golden Sand)
- **Fonts**: Gloock (headings), Merriweather Sans (body)

### Features
- **Hero Carousel**: Auto-rotating image gallery with 20-second intervals and manual navigation controls
- **Responsive Design**: Mobile-first CSS Grid layout with auto-fit columns
- **Progressive Navbar**: 140px height with enlarged branding
- **Content Sections**: 
  - Homepage with service times and testimonials
  - What's On page with groups, organisations, and event calendar
  - Visit Us page with directions and visitor information
  - Room Hire page with facility details and booking information
  - About & Contact pages

### Key Pages
- `index.html` - Homepage with hero carousel and key information
- `whats-on.html` - Events, groups, organisations, and weekly schedule
- `room-hire.html` - Room facilities and hiring information
- `visit.html` - Visitor information and directions
- `about.html` - Church information
- `contact.html` - Contact details and enquiry form

### Technical Stack
- HTML5 with semantic markup
- CSS3 with custom properties (design tokens)
- Vanilla JavaScript for carousel functionality
- Google Fonts integration
- WebP image format
- Fully accessible with ARIA labels and skip links

### Development
Run local server:
```bash
python3 -m http.server
```
Visit: `http://localhost:8000`
