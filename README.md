# stmary

St Mary's is a small, friendly church with a big heart! We're situated in Handforth village, East Cheshire. We are a part of the Bramhall & Wythenshawe Methodist Circuit in the North West of England District.

## Getting Started in VS Code

### 1. Clone the repository

```bash
git clone https://github.com/TopG85/stmary.git
cd stmary
```

### 2. Open in VS Code

```bash
code .
```

Or open VS Code, choose **File → Open Folder…**, and select the `stmary` folder.

### 3. Install recommended extensions

When VS Code opens the project it will prompt you to install the recommended extensions. Click **Install All**, or open the Extensions panel (`Ctrl+Shift+X` / `Cmd+Shift+X`), search for `@recommended`, and install them. Key extensions include:

| Extension | Purpose |
|---|---|
| **Live Server** | One-click local dev server with live browser reload |
| **Prettier** | Consistent code formatting for HTML, CSS and JS |
| **Auto Rename Tag** | Automatically renames paired HTML tags |
| **Auto Close Tag** | Automatically closes HTML tags as you type |
| **HTML CSS Support** | IntelliSense for CSS class names in HTML |
| **Code Spell Checker** | Catches spelling mistakes in content and code |

### 4. Preview the site locally

After installing the **Live Server** extension, right-click `index.html` in the Explorer panel and choose **Open with Live Server**, or click the **Go Live** button in the bottom status bar. The site will open in your browser and automatically reload whenever you save a file.
